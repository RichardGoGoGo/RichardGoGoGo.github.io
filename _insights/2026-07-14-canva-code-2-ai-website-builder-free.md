Canva 宣布将 Code 2.0 功能向全部 2.65 亿月活跃用户（含免费账户）开放：用户通过自然语言提示即可生成完整的响应式网站或可运行应用，支持 HTML 导入、50 余套模板，从提示到上线时间压缩 30%，生成速度较前版本提升 75%。

> Canva 是视觉设计领域用户基础最大的工具之一，Code 2.0 免费化意味着无编程基础的设计师可直接将视觉方案转为可运行网站原型，进一步压低了 AI 辅助原型制作的门槛。
