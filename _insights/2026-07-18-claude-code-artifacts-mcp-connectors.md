Anthropic 于 7 月 16 日为 Claude Code Artifacts 上线 MCP 连接器调用能力：生成的看板、仪表盘等页面可直接连接实时数据，不同查看者按各自权限拉取不同内容；页面还可内嵌操作按钮（如发送 Slack 消息、更新 Issue），从静态展示页升级为动态数据面板。

> Artifacts 从静态展示转向动态数据产品，为快速搭建可运行数据可视化与交互原型提供了更直接的路径，「原型即产品」的实现门槛进一步降低。
