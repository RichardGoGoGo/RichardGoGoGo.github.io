Anthropic于2026年7月16日上线Claude Code Artifacts的MCP连接器调用能力，生成的看板或仪表盘页面可连接实时数据——不同查看者按各自权限拉取不同内容；页面还可嵌入操作按钮（如发送Slack消息或更新GitHub Issue），将Artifacts从静态展示转变为动态数据面板。该功能由新智元于7月18日报道。

> Artifacts接入实时数据与操作按钮，展示了Claude Code将设计方案转变为可运行实时数据产品的路径，是「设计即产品」方向的代表性功能更新。
