创作者阿真于 2026 年 6 月 23 日在 GitHub 开源 comic-ip-onboarder-skill，这是一个由 Codex Agent 引导的 IP 设计自动化工具，输入 IP 原图后可自动生成三视图、表情包系列、信息图等衍生内容，实现 IP 形象的标准化全套产出；工具已完整开源，设计师和学生可直接克隆部署使用。

> 将 AI Agent 引入 IP 衍生设计全流程，是 AI 辅助创意量产的直接落地案例；开源且可本地部署的特性，使其成为品牌设计、插画设计教学课堂中可实际操作的实验工具。
