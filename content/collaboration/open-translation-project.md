## 项目概述

「浅译万道」发起开源翻译协作计划，旨在将 AI 领域的优质英文技术文档翻译为高质量中文版本，降低中文社区的学习门槛。

翻译范围包括但不限于：

- AI 平台官方文档（OpenAI、Anthropic、Google 等）
- 经典研究论文的通俗解读
- 技术博客与教程

## 参与方式

### 如何加入

1. 在 GitHub 仓库中认领待翻译文档
2. Fork 仓库，在 `drafts/` 中完成初稿
3. 使用 STYLE-GUIDE.md 中的规范整理格式
4. 提交 Pull Request 进行审核

### 角色分工

| 角色 | 职责 |
|------|------|
| 译者 | 完成初稿翻译 |
| 审校 | 术语一致性与准确性审核 |
| 排版 | Markdown 格式与 front matter 规范化 |
| 维护 | 跟踪原文更新，同步译文 |

## 当前进展

- 已建立翻译规范与工作流程
- 已完成示例翻译：OpenAI 提示工程指南
- 正在招募首批翻译志愿者

## 计划路线

| 阶段 | 目标 | 时间 |
|------|------|------|
| 第一期 | 完成 5 篇核心文档翻译 | 2026 Q2 |
| 第二期 | 扩展至 20 篇，建立审校机制 | 2026 Q3 |
| 第三期 | 开放社区投稿，自动化工作流 | 2026 Q4 |
