2026 年 6 月底，Agnes AI 发布 PC 端创作平台 Pavo，被多家媒体称为首个真正免费、不限时的视频、短剧与图像创作平台。其核心是把"拆剧本 → 角色设计 → 分镜 → 成片"整合进一条稳定工作流，用户只需一句话描述，平台即可自动完成多步生成，而无需在多个工具间来回切换。据报道，其全模态模型 API 单周调用量达 4.66 万亿 Token。

## 已知事实

- **平台与模块**：Pavo 为 PC 端 AI 图像、视频与短剧创作平台，整合短剧、Agent、图像生成、视频生成四大功能模块。
- **智能路由 Harness**：内置名为 Harness 的智能调度系统，可把创作者的一句话请求自动拆解为剧本、角色设计与分镜，由多个专业 Agent 像导演、编剧、分镜师一样协作完成。
- **自研全模态模型**：自 6 月 1 日起，Agnes 开放其核心多模态模型 API，供不限量免费使用，包括文本模型 Agnes-2.0-Flash、图像模型 Agnes-Image-2.1-Flash、视频模型 Agnes-Video-2.0；并预告即将发布的 Agnes-Video-2.5-Preview 在运动连贯性、角色一致性、运镜与背景上有明显提升。
- **市场切入**：多家媒体将其解读为，在 AI 短剧制作日益烧钱的背景下，Agnes 以"免费"作为切入策略。

## 横向与纵向定位

横向看，当前主流的 AI 视频与短剧生成多采用按量计费（按生成次数或积分扣费），对需要反复试错、批量出片的创作者构成持续成本。Pavo 的差异点有二：一是把"自研全模态模型 + API 永久免费不限量"作为打法，直接降低试错成本；二是用 Harness 这类智能路由把"一句话到成片"的多步流程串成一条龙，而非只提供单点的"文生视频"能力。前者是商业模式上的差异，后者是工作流编排上的差异。

纵向看，AI 视频生成正从"单段画面生成"走向"由智能体编排的完整创作工作流"。早期的关注点是单次生成的画质与时长；而 Pavo 这类产品把重心放在"拆解—分工—串联"——让多个 Agent 分别承担编剧、角色、分镜等职责，把短剧制作的流程整体自动化。叠加"免费"的商业策略，反映出 AI 短剧赛道在工具成熟后开始向生态争夺与降本方向竞争。

> ⚠️ 待核实："永久免费、不限量"的可持续性与背后的商业模式（如何覆盖算力成本）未充分披露；"单周 API 调用 4.66 万亿 Token"为报道数据，未见独立核验；Agnes AI 的公司背景、团队与融资情况，本文未从一手信源逐项核到；本文主源量子位页面本次未能完整抓取，事实据多源报道综合，具体以官方为准。

> Pavo 把"一句话到成片"的短剧创作流程交由智能体编排，并以自研全模态模型 API 永久免费切入烧钱的 AI 短剧赛道，是 AI 视频从"单段生成"走向"完整工作流自动化"、并向降本与生态争夺演进的代表性动作；对影视、动画、数字媒体方向的创作教学与研究，提供了一个低门槛、可观察的 AI 短剧全流程样本。
