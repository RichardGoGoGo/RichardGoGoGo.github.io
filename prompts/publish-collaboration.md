# 合作交流发布提示词

## 任务
将 `drafts/` 中的合作交流原稿整理为标准化 Markdown，输出到 `content/collaboration/`。

## 输入
- 原稿文件：`drafts/{filename}`
- 参考模板：`templates/collaboration.md`
- 转换规范：`STYLE-GUIDE.md`

## 输出
- 文件路径：`content/collaboration/{slug}.md`
- 文件格式：标准 Markdown + YAML front matter

## 要求
1. 按照 `templates/collaboration.md` 的章节结构组织内容
2. front matter 中 category 设为 `collaboration`
3. 清晰说明参与方式和联系渠道
4. 保留项目的具体进展信息
5. 不虚构合作方或进展数据
6. 遵循 STYLE-GUIDE.md 的所有格式与语言规范
