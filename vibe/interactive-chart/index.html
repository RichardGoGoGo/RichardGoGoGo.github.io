高级信息图与可视化类型图鉴

按「流程与关系 / 层级与系统 / 变化·比较·多维」三类，给每种高级图表一个可交互的实时示例。

首页/ VIBE工具箱/ 高级信息图与可视化类型图鉴

图表是否「高级」，取决于它是否准确表达了关系，而非名称是否复杂。以下每张为演示生成的示例数据，鼠标悬停可查看数值与路径。先按顺序做每类前两种。

一 · 流程与关系
桑基图
Sankey Diagram
定量

用不同宽度的带状连线表示数量在多个节点间的流动与分配，连线越宽流量越大。

冲积图
Alluvial Diagram
定量·半定量

展示一组对象经过多个阶段时，分类如何变化、分流与延续，适合观察群体迁移。

和弦图
Chord Diagram
定量

将多个对象排列在圆周上，用弧形带连接彼此，表现多对多关系及其强弱。

力导向网络图
Force-Directed Network
半定量·概念

用引力与斥力模拟形成网络布局，识别中心节点、聚类与连接密度。

弧线关系图
Arc Diagram
半定量·概念

节点按顺序排成一条直线，用跨越节点的弧线表示关系，适合展示跨阶段连接。

因果回路图
Causal Loop
概念

用带方向的连线表示促进、抑制与反馈，解释系统为何增强、衰减或形成循环。

服务蓝图
Service Blueprint
概念

把客户动作、前台、后台、支持放在同一时间轴，展示一项服务如何被完整交付。

泳道图
Swimlane
概念

按角色划分泳道，展示各方在同一流程中的动作、责任与交接。

状态机图
State Machine
概念

展示对象可能的状态、触发事件与状态变化，以及异常、退回与终止分支。

用户旅程地图
Journey Map
半定量·概念

沿时间线展示用户在各阶段的行为、目标、接触点与体验变化。

平行集合图
Parallel Sets
定量

把多个分类维度平行排列，用带状连线展示类别组合之间的数量与路径。

地铁线路图
Metro Map
概念

借用地铁线路的视觉语言，把多条路径、阶段与交汇点组织成易追踪的系统。

分支时间线
Branching Timeline
半定量·概念

在时间轴上表现主线、支线、关键节点与不同选择造成的后续路径。

径向流程图
Radial Process
概念

沿圆环安排阶段，适合表达循环、复盘、持续运营与多轮迭代。

二 · 层级与系统结构
旭日图
Sunburst Chart
定量

用同心圆环展示从中心向外的多层级结构，扇区角度对应数量或占比。

矩形树图
Treemap
定量

用嵌套矩形同时表达层级与规模，矩形面积代表数量，颜色区分大类。

圆形打包图
Circle Packing
定量

用大小不同、相互嵌套的圆表示类别、子类及规模，视觉比矩形树图更柔和。

冰柱图
Icicle Chart
定量·半定量

用逐层展开的条带展示层级关系，适合追踪一个大类如何拆分为多个子项。

树状图
Dendrogram
半定量·概念

用逐级分叉的树结构展示分类、相似性、聚类过程或组织层级。

能力地图
Capability Map
概念

把组织或系统的能力按领域与层级排列，强调「能做什么」及能力归属。

分层系统图
Layered System Map
概念

按界面、流程、执行、数据、基础设施等层级展示系统的组成与上下游关系。

爆炸轴测图
Exploded Axonometric
概念

把系统各层沿空间方向拆开，既保留整体结构，又能看清每层的组成与连接。

轨道图
Orbit Diagram
概念

以核心对象为中心，在多层轨道上安排相关角色、能力与资源，表现中心与外围关系。

蜂巢能力图
Honeycomb Map
概念

用相邻六边形组织多个平级模块，适合展示能力组合、服务模块与产品体系。

维诺图
Voronoi Diagram
定量·半定量

按各中心点的影响范围划分不规则区域，表现领域分区、归属与空间邻近关系。

维诺树图
Voronoi Treemap
定量

将层级与规模编码放入不规则有机区域，在保持比例的同时获得更自然的视觉效果。

星座图
Constellation Map
半定量·概念

以少量核心节点与大量外围节点构成类似星座的关系图，强调主题、关联与聚类。

剖面图
Cutaway Diagram
概念

通过切开外层展示内部结构与隐藏部件，常用于系统、空间或产品机制说明。

三 · 变化 · 比较 · 多维
小多图
Small Multiples
定量·半定量

用一组结构一致的小图展示不同对象或时段，便于横向比较变化与模式。

版本故事板
Iteration Storyboard
概念·半定量

按时间顺序展示一个方案的多个版本，并标注每轮的选择、修改与原因。

平行坐标图
Parallel Coordinates
定量

把多个指标画成平行坐标轴，每个对象用一条折线穿过各轴，适合比较多维特征。

热力矩阵
Heatmap Matrix
定量·半定量

用颜色深浅表达两个维度交叉后的数值、强度、频率或风险等级。

流图
Streamgraph
定量

用上下起伏的连续色带展示多个类别随时间的数量变化，整体形态类似河流。

隆起图
Bump Chart
定量

用多条线展示不同对象的排名如何随时间变化，重点观察上升、下降与交叉。

山脊图
Ridgeline Plot
定量

将多组分布曲线层叠排列，便于比较不同群体、时间或条件下的数据形态。

蜂群图
Beeswarm Plot
定量

把每个数据点单独显示并错位避免重叠，既看总体分布，也保留个体。

马赛克图
Marimekko / Mosaic
定量

通过宽度与高度同时表达两个比例维度，适合比较总量与内部构成。

日历热力图
Calendar Heatmap
定量

将每天的数据映射到日历网格，用颜色表示活跃度、频率或完成量。

坡度图
Slopegraph
定量

用简洁连线比较多个对象在两个时间点或两种条件下的数值变化。

子弹图
Bullet Chart
定量

在紧凑空间内同时表示实际值、目标值与绩效区间，适合替代仪表盘。
